# Vault 开始使用

## 最简单的方式

双击仓库里的 `点我开始.cmd`。

打开窗口后按这个顺序操作：

1. 点击“选择...”，选择一个专门用来归档的文件夹。
2. 点击“初始化归档”。
3. 点击“打开放文件的文件夹”，把要保存的文件放进去。
4. 点击“预演维护”，先看看将要执行的操作。
5. 点击“正式维护”，完成快照、同步、校验和必要修复。

以后每隔几个月打开一次助手，点击“预演维护”和“正式维护”即可。

新增文件可以直接维护。修改已有文件并确认内容正确后，先点击“同步已确认更改”，再运行维护。`maintain` 检测到已有文件被删除或内容变化时，会在改动镜像、快照和 baseline 前停止。

## 文件应该放在哪里

只把文件放进助手打开的“放文件的文件夹”。这个文件夹实际路径是：

```text
你的归档文件夹\master
```

不要手动修改这些文件夹：

```text
你的归档文件夹\mirrors
你的归档文件夹\snapshots
你的归档文件夹\.archive
```

它们由 Vault 自动维护。

## 仓库和归档文件夹的区别

`D:\Vault` 是工具本身。

真正要长期保存的文件应该放在你选择的归档文件夹里，例如：

```text
D:\我的归档\master
```

## 需要看结果时

在助手里点击：

- “查看状态”：查看当前文件数量和归档状态
- “打开最近报告”：打开最近一次运行生成的摘要报告
- “打开报告文件夹”：查看所有历史报告
